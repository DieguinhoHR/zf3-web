### Description

zf create controller creates a controller file where the leading slash is missing on the extends section.

i.e. it creates:

 
    <pre class="highlight">
    namespace Core;
    
    class IndexController extends Zend\Controller\Action
    {


and should be:

 
    <pre class="highlight">
    namespace Core;
    
    class IndexController extends \Zend\Controller\Action
    {
