### Description

Manual entry #42.2 contains incorrect example of creating new queue. Method _create_ does not exists. _createQueue_ method should be used instead of _create_ [http://framework.zend.com/manual/en/â€¦](http://framework.zend.com/manual/en/zend.queue.example.html)

 
    <pre class="highlight">
    // Create a new queue
    $queue2 = $queue->create('queue2');


Correct code

 
    <pre class="highlight">
    // Create a new queue
    $queue2 = $queue->createQueue('queue2');
