### Description

The comments say:

 
    /**
     * Returns value if it is a valid US ZIP, FALSE otherwise.
     *
     * @param mixed $value
     * @return mixed
     */


But looking at the code it will return a boolean.

 
    public static function isZip($value)
    {
        return (bool) preg_match('/(^\d{5}$)|(^\d{5}-\d{4}$)/', $value);
    }


And so it does:
