### Description

There is a bug in the example script in [http://framework.zend.com/manual/en/â€¦](http://framework.zend.com/manual/en/zend.view.scripts.html) about how to instanciate Zend\_View\_Smarty class. Code error:

 
    <pre class="highlight">
    //Example 1. In initView() of initializer.
    $view = new Zend_View_Smarty('/path/to/templates');
    $viewRenderer =
        new Zend_Controller_Action_HelperBroker::getStaticHelper('ViewRenderer');

    <pre class="highlight">
    $viewRenderer = Zend_Controller_Action_HelperBroker::getStaticHelper('ViewRenderer');
