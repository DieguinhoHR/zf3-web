 Issue Type: Coding Standards Violation Created: 2006-11-03T21:32:09.000+0000 Last Updated: 2007-07-05T14:43:25.000+0000 Status: Resolved Fix version(s): - 0.6.0 (16/Dec/06)
 
 Reporter:  Matthew Ratzloff (mratzloff)  Assignee:  Ralph Schindler (ralph)  Tags: - Zend\_Session
 
 Related issues: 
 Attachments: 
### Description

Many properties and methods use underscores instead of camelCaps, which is the Zend Framework standard. Except for the php.ini directives, consider changing these to camelCaps.
