ZF-7076: Edit links always used when uploading files to multipart MIME Zend\_Gdata services
-------------------------------------------------------------------------------------------

 Issue Type: Bug Created: 2009-06-21T20:43:42.000+0000 Last Updated: 2009-06-21T21:02:20.000+0000 Status: Resolved Fix version(s): - 1.8.4 (23/Jun/09)
 
 Reporter:  Trevor Johns (tjohns)  Assignee:  Trevor Johns (tjohns)  Tags: - Zend\_Gdata
 
 Related issues: 
 Attachments: 
### Description

When uploading files to services requiring multipart-MIME (YouTube, Picasa), the media edit link is always used, even if a different URL is provided by the user.
