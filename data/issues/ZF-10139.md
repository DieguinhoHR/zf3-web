### Description

In the Programmer's Reference Guide ([http://zendframework.com/manual/1.10/â€¦](http://zendframework.com/manual/1.10/en/zend.controller.router.html)) the lighttpd rewrite rule there is:

    url.rewrite-once = (
        ".<em>\?(.</em>)$" => "/index.php?$1",
        ".*.(js|ico|gif|jpg|png|css|html)$" => "$0",
        "" => "/index.php"
    )

Since 1.4.24, lighttpd adds a new configuration option that we can use to more closely emulate the Apache .htaccess behavior:

    url.rewrite-if-not-file = (
        "/(.*)$" => "/index.php?$1",
    )
