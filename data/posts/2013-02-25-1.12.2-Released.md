 The Zend Framework community is pleased to announce the immediate availability of Zend Framework 1.12.2! Packages and installation instructions are available at:

- [http://framework.zend.com/downloads/latest](/downloads/latest)

<!--more-->

Changelog
---------

 This release includes almost 50 bugfixes and stability fixes. In particular, a fix was released for `Zend_Service_Twitter`, ensuring that it will continue to work with the Twitter API going forward. _If you are using the `Zend_Service_Twitter` component, please upgrade immediately, or your code will not work!_

 To see the complete set of issues resolved for 1.12.2, please visit the changelog:

- [http://framework.zend.com/changelog/1.12.2](/changelog/1.12.2)

Thank You!
----------

 Many thanks to all contributors to this release!
